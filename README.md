## 🎯 問題設定

### 1. アイデアロス（Idea Loss Phenomenon）
AI時代に大量生成されるアイデアの多くは、会話が終わると消えてしまいます。人類史上最大規模のアイデア・思考の揮発現象が起きています。Cyvaiはそれらを定着させ、**“思想データ”**としてアーカイブします。

### 2. AGI の燃料不足
AIが真に渇望しているのは、整然とした正解データではありません。「なぜそう思ったのか」という人間の生々しい理由・感情・文脈のデータ群です。

## 🛠️ ソリューション：2層のアーキテクチャ
Cyvaiは、ユーザーを集めるための**「アイデアプラットフォーム（表面）」**と、そこから価値を抽出する**「思想インフラ（深層）」**の2層構造で機能します。

### 【表面レイヤー】 アイデア流通プラットフォーム
ユーザー向けに、アイデアを価値化・流通させるUXを提供します。

* **2クリック投稿**: ChatGPTやClaudeでの会話中に「まとめて」と依頼し、専用ボタンからスムーズに投稿。AIが自動でタイトル・タグ付け等を行います。
* **90日ルール (流動性の強制)**: 投稿から90日間は独占的な買収交渉期間とし、期間を過ぎれば自動的にフリー化（公開）されます。企業側に「他社に取られる前・公開される前に買う」という決断を促し、知的財産の流動性を極限まで高めます。
* **多次元評価とAIマッチング**: 「💎 儲かりそう！(市場性)」「🧠 天才的！(革新性)」など、多様な軸でレビュー。この多層的な評価データが蓄積されるほどAIのマッチング精度が向上します。
* **独自の評価経済圏**: ライト層（ひらめきを投げる）とプロ層（事業化する）が交差するコミュニティ。評価や派生アイデアの創出自体にインセンティブを与えます。
* **自動契約**: 事前に包括的な同意を行うことで、対人折衝なしでアイデアの売買が自動成立。マネタイズのハードルを極限まで下げます。

### 【深層レイヤー】 思想インフラとしての本質的価値
アイデア売買からの手数料収益は二次的なものです。本当の価値は、プラットフォーム内で集約された**思想データそのもの**にあります。

1.  **評価レイヤー (Evaluation Layer)**: ユーザーの判断・選択・迷いのプロセスをデータ化し、蓄積。
2.  **多元性 (Multiplicity)**: 単一の軸（Good/Bad）ではなく、倫理、未来性、創造性など、人間の判断の多元性をそのままデータ化。
3.  **不完全さの体系化 (Crap Data Architecture)**: 迷い、誤解、勘違い。削ぎ落とされがちなノイズの中にこそ、AIの人格形成に不可欠な学習素材が眠っています。

> **ビジネスモデルの真髄** > 在野のアイデアマンやクリエイターから得られる「生の人間データ・文脈データ」を、AI企業への学習基盤として提供することが本来の目的です。
